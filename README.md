# SCE Moodle Assignment Tracker 🎓

**הדרך החכמה והקלה ביותר לנהל את כל המטלות שלך במכללת סמי שמעון (SCE).**

SCE Moodle Assignment Tracker הוא תוסף כרום קל ומהיר שמתחבר ישירות לחשבון ה-Moodle שלכם בצורה מאובטחת. בלחיצת כפתור אחת, הוא סורק את כל הקורסים שאליהם אתם רשומים, מושך את כל המטלות הפתוחות, ומציג לכם אותן בלוח בקרה (Dashboard) מודרני, מעוצב ונוח.

## פיצ'רים מרכזיים ✨
* **לוח בקרה חכם:** כל המטלות מכל הקורסים מרוכזות במסך אחד.
* **זיהוי דחיפות ויזואלי:** התוסף מחשב את הזמן שנותר להגשה ומציג התרעות צבעוניות (אדום לדחוף, צהוב לקרוב, וירוק לרחוק).
* **חלוקה הגיונית:** הפרדה בין עבודות שעדיין פתוחות (Pending) לבין עבודות שכבר הגשתם.
* **סינון לפי קורסים:** לחיצה על שם של קורס תסנן את התצוגה למטלות שלו בלבד.
* **גישה מהירה:** לחיצה על כל מטלה תעביר אתכם ישירות לעמוד ההגשה ב-Moodle.

---

## Privacy Policy 🔒
This extension is built with strict privacy principles in mind:
* **No External Servers:** All processing and data storage occurs locally within your browser. 
* **No Data Collection:** We do not collect, transmit, store, or share any personal information, Moodle credentials, or assignment grades.
* **Functionality:** The extension acts purely as a local tool to read assignment deadlines from your active Moodle session and display them in a user-friendly interface.

By using this extension, you can be assured that your academic data remains completely private and secure on your own machine.
